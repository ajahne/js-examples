- timers (e.g. setTimeout, setInterval)
- loops
- strings
- IFFE
- revealing module pattern
- promises
- data structures (from scratch)
- sorting algorithms (from scratch)
- ES6
  - const, let
  - block scope
  - arrow functions
  - modules
  - classes
  - async
